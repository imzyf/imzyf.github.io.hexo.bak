在使用 `git status` 时，中文文件名出现乱码
<!--more -->
```
git config --global core.quotepath false
```
`core.quotepath` 设为 `false` 的话，就不会对 `0x80` 以上的字符进行 quote，中文显示正常

> - [Git中文显示问题解决](http://xstarcd.github.io/wiki/shell/git_chinese.html)
