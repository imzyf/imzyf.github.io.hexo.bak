指定本次提交时间：

```bash
git commit -m "fix..." --date=`date -R`

git commit -m "fix..." --date="$(date -R)"

git commit -m "fix..." --date="Tue, 11 Jun 2019 17:50:50 +0800"
```

<!-- more -->

修改上次提交时间：

```bash
git commit --amend --date=`date -R`
```
