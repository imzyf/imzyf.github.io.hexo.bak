## 编辑 settings.json

`command + ,` 右上角点击 `打开设置`。

```json
// 通过整体放大窗口比例
"window.zoomLevel": 1.1,
// 字体
"editor.fontSize": 14,
// 软换行
"editor.wordWrap": "on",
```
