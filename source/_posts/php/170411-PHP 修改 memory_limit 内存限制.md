在进行大量统计运算时，PHP 可能产生内存不足的情况。

## 查看 memory_limit

```php
<?php phpinfo(); ?>
```

## 修改 memory_limit

可以在 `phpinfo()` 中找到 `Loaded Configuration File` `php.ini` 的位置

修改 `php.ini` 中的 memory_limit

```ini
memory_limit = 1024M;
```

如果没有，你可以在文件的尾部自己增加这个参数
