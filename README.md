# ZYF.IM 源文件

## 初始化环境

```bash
yarn install hexo-cli -g
yarn
hexo g
git init .
git remote add origin git@github.com:imzyf/imzyf.github.io.git
```

## Tips

按 `t` 进入搜索模式。

## Hexo command

```bash
# generate
hexo g

# server
hexo s
```
