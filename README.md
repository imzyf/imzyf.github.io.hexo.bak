# https://ZYF.IM 源文件
## 初始化环境
```bash
npm install hexo-cli -g
yarn add
mkdir public && cd public
git init .
git remote add imzyf root@zyf.im:/home/yifan/project/imzyf
```

## Hexo command
```bash
# generate
hexo g

# server
hexo s
```
